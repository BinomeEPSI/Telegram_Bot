## Installation

`$ npm install`

```bash
$ cp .env.sample .env
$ vim .env
# Fill TOKEN variable.
```

## Heroku setup

`$ heroku git:remote -a HEROKU_APP_NAME`

Whenever you want to deploy

`$ git push heroku master`

## Run

### Tests

`npm test`

### Bot (local)

`npm start`
